# Product Overview

## Simple Instagram Clone

A social media web application that replicates core Instagram functionality, allowing users to share photos and interact through likes and comments.

## Core Features

- **User Authentication**: Registration and login system with session management
- **Photo Sharing**: Upload images with captions and descriptions
- **Social Feed**: Chronological timeline of all user posts
- **Social Interactions**: Like and comment on posts
- **User Profiles**: Personal pages showing user information and their posts

## Target Users

Individual users who want to share photos and engage with others in a social media environment.

## Key User Flows

1. **Registration/Login** → **Feed** → **Post Upload** → **Social Interaction**
2. **Browse Feed** → **Like/Comment** → **View Profiles**
3. **Upload Content** → **Manage Profile** → **View Own Posts**

## Business Logic

- Posts are displayed in reverse chronological order (newest first)
- Users can only edit/delete their own content
- All interactions require authentication
- Image uploads are validated for format and size
